# Dashboard QA Rubric

Run this quick rubric before sharing final Tableau links.

- [ ] **Accuracy**: KPI values match processed dataset filters.
- [ ] **Clarity**: Titles and axis labels are specific and readable.
- [ ] **Interactivity**: Filters and actions update all intended views.
- [ ] **Performance**: Dashboard loads smoothly with extract enabled.
- [ ] **Narrative flow**: Executive summary leads into diagnostic views.
- [ ] **Accessibility**: Colors are distinguishable and font sizes are readable.
